- Equation:y=mx+c
- Use Case: Predicting income based on years of experience.

- Definition: Extends linear regression to use two or more independent variables to predict the dependent variable.
- Equation:y=m1x1 + m2x2 + m3x3 + ........... + c
- Use Case: Predicting income based on age and experiencs.
